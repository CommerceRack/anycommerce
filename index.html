Create AWESOME transactional e-commerce and native ios/android Rich-Internet-Single-Page-Applications using simple HTML5 + CSS3 + JS/jQuery.

Why you should use this:

  • Improved UX: User experience is improved instantaneous page loads and creative transitions of content.
  • Designer Friendly: The "TLC" code patterns used within the framework make it easy for end-users who are HTML novices to maintain and make changes.
  • SEO Friendly: CommerceRack backend supports 100% customizable escaped fragment syntax for unbeatable page indexing.
  • Cloud Ready: Static app files are easy to deploy at the network edge using CDN services like Amazon CloudFront.
  • Native Shells: using tools like Apache Cordova the apps can be published directly into native app stores, and leverage device capabilies like cameras, wallets and location.
  • Mobile/WiFi Friendly: Apps making small compressed json requests don't use nearly s much bandwidth as conventional websites, and they never need to wait for a page to load.
  • Reduced Infrastructure: Apps use extensive client side processing (eliminating the need for giant backend servers) within a native android/ios shell, or in a browser effectively running the "web server" directly on the users device.
  • Improved Security: CommercerRack uses a single api + access control layer which is completely separated from the display logic. This results in a significantly reduced threat profile. Common attacks like XSS/CSRF are ineffective against apps due to the fact they run locally on a device.
  • Rapid Response: Intelligent Caching by the CommerceRack can determine frequently used data and make sure stored in a hot-object cache. In addition hot objects can be synchronized and cached locally on apps, saving bandwidth, and i/o. This unique approach to caching significantly reduces costs in environments like AWS where these services are billable.
  • Faster Development: Get more done in less time, supports local testing (no separate test environment needed). Uses GIT(hub) for change control, and user/multi-vendor collaboration.
  • Freedom: 100% Open Source, MIT licensed.
